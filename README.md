# Nearbuy

Singapore business directory built with Vite, React, TypeScript, shadcn-ui, and Tailwind CSS.

## Getting started

```sh
npm install
npm run dev
```

The local development server runs at `http://localhost:8080/`.

## Available scripts

- `npm run dev` starts the Vite development server.
- `npm run build` creates a production build.
- `npm run build:dev` builds using the development mode config.
- `npm run lint` runs ESLint.
- `npm run test` runs the Vitest suite once.
- `npm run test:watch` runs Vitest in watch mode.
